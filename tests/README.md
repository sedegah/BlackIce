# BlackIce Test Assets

## Structure

- `pcap/` replay fixtures (not committed by default)
- `messages/feature_windows/` canonical JSON feature windows for quick IPC checks
- `logs/` output directory for smoke/perf scripts

## Quick start

Run all automated checks:

```bash
./scripts/run_test_matrix.sh
./scripts/run_test_matrix.ps1
scripts\run_test_matrix.cmd
```
